# 🎰 Casino-Club F2P: 최종 시스템 검증 가이드

이 문서는 재구축 및 안정화 작업이 완료된 Casino-Club F2P 프로젝트의 모든 기능이 정상적으로 작동하는지 확인하기 위한 최종 검증 절차를 안내합니다.

## 1단계: 시스템 실행

터미널을 열고, 프로젝트의 루트 디렉토리에서 다음 명령어를 실행하여 모든 서비스를 시작합니다.

```powershell
.\\docker-manage.ps1 start --tools
```
이 명령어는 백엔드, 프론트엔드, 데이터베이스 및 모든 개발 도구 컨테이너를 실행합니다. 서비스가 모두 정상적으로 시작될 때까지 잠시 기다려주십시오. `docker-compose ps` 명령어로 모든 컨테이너의 `Status`가 `Up` 상태인지 확인할 수 있습니다.

## 2단계: 자동 통합 테스트 (E2E)

시스템이 실행 중인 상태에서, 다음 명령어를 실행하여 제가 작성한 **전체 사용자 여정 테스트 스크립트**를 실행합니다.

```powershell
python test_api_integration.py
```

### ✅ 예상 성공 결과
스크립트가 실행되면서, 다음과 같이 각 기능 API가 순서대로 호출되고 테스트되는 과정이 출력됩니다.

```
🚀 전체 사용자 여정 E2E 테스트 시작...
✅ 헬스체크 통과...
--- 1. 온보딩 ---
✅ 회원가입 통과...
✅ 프로필 조회 통과...
--- 2. 게임 플레이 ---
✅ 슬롯 게임 통과...
✅ 룰렛 게임 통과...
✅ 가챠 뽑기 통과...
--- 3. 결과 확인 ---
✅ 보상 내역 조회 통과...
--- 검증 ---
총 3개의 보상 기록이 확인되었습니다.

🎉 전체 사용자 여정 테스트 통과!
```
최종적으로 **"🎉 전체 사용자 여정 테스트 통과!"** 메시지가 출력되면, 백엔드의 모든 핵심 기능이 유기적으로 완벽하게 작동함을 의미합니다.

## 3단계: 수동 기능 검증 (UI/UX)

자동 테스트 성공 후, 다음 시나리오에 따라 프론트엔드 UI에서 직접 기능을 검증합니다. 이 과정에서 **기존의 모든 디자인과 레이아웃이 보존되었는지** 확인하는 것이 중요합니다.

1.  **회원가입:**
    *   브라우저에서 `http://localhost:3000`에 접속합니다.
    *   회원가입 페이지로 이동하여, 초대코드에 **`5858`**을 입력하고 새로운 계정을 생성합니다.
2.  **로그인 및 프로필 확인:**
    *   방금 생성한 계정으로 로그인합니다.
    *   프로필 페이지로 이동하여, 초기 지급된 토큰과 자신의 닉네임이 정확히 표시되는지 확인합니다.
3.  **게임 플레이:**
    *   **가위바위보, 슬롯, 룰렛** 게임 페이지로 각각 이동합니다.
    *   코인을 베팅하고 게임을 한 판 플레이합니다.
    *   게임 결과(승/패/무)에 따라 토큰 잔액이 실시간으로 변경되는지 확인합니다.
4.  **보상 내역 확인:**
    *   '리워드 내역' 페이지로 이동합니다.
    *   방금 플레이한 게임의 결과(획득 또는 잃은 코인)가 새로운 기록으로 추가되었는지 확인합니다.
5.  **관리자 기능 확인 (선택 사항):**
    *   `pgAdmin` (`http://localhost:5050`)에 접속하여 `users` 테이블에서 특정 사용자를 `is_admin = true`로 변경합니다.
    *   해당 사용자로 다시 로그인한 후, `/api/admin/users` 등의 API를 직접 호출하여 관리자 기능이 정상적으로 작동하는지 확인합니다.
